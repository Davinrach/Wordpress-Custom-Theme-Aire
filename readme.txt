== Description ==

Airemodern is a modern WordPress block theme designed for performance and flexibility. It leverages the full power of the Site Editor (FSE).

== Installation ==

1. In your WordPress dashboard, go to Appearance > Themes > Add New.
2. Upload the `airemodern` folder to the `/wp-content/themes/` directory.
3. Activate the theme through the 'Themes' menu in WordPress.
